About Rico

This website is currently under development, just a small side project I have. Website is hosted on GitHub, and domain was acquired from the wonderful people at NameCheap. I am using the Grayscale bootstrap theme which you can see at the preview page. The theme is open source, and you can use it for any purpose, personal or commercial.

This theme features stock photos by Gratisography along with a custom Google Maps skin courtesy of Snazzy Maps.

Grayscale includes full HTML, CSS, and custom JavaScript files along with LESS files for easy customization.

One thing that I learned so far is that Jekyll (which GitHub uses) (also not sure what it does) ignores anything in folders named node_modules or vendor, which was causing issues; renamed 'vendor' folder to 'res'.
